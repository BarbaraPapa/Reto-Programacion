Dia 12

Crea un slider que permita seleccionar un número entre 1-50 y lo muestre en vivo al cambiar.

25

Para crear un slider con html podemos usar la etiqueta < input > con type="range" donde también podemos insertar un mínimo(min) y un máximo(max), por defecto los valores van de 0 a 100.

Podemos agregar un < output > para agregar el valor del < input > y actualizarla usando una sola línea de Javascript onInput="this.nextElementSibling.value = this.value"

<input type="range" min="1" max="50" onInput="this.nextElementSibling.value = this.value" /> <output> </output>
