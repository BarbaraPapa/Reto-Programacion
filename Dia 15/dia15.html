Crea una página con un video MP4 (no de youtube), que muestre una imagen de portada antes de darle a reproducir.

Usamos la etiqueta < video > para insertar un video dentro de una página web con el atributo src que especifica la ruta del archivo de video que se mostrará. Usamos el atributo poster para especificar la imagen que se mostrará como vista previa del video antes de reproducirlo. Agregamos el atributo controls para mostrar los controles estándar para el reproductor de video, como el botón de reproducción/pausa, el volumen y la barra de progreso.

<video src="video.mp4" poster="picture.jpg" controls></video>
