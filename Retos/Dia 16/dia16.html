⬅ Volver

Para mostrar un texto con el atajo de teclado se usa la etiqueta
< kbd > (keyboard)

Las funciones que podemos realizar con Ctrl + Alt + Supr que realizamos con nuestro teclado puede contener distintas funcionalidades, las cuales dependerá del procedimiento que queramos llevar a cabo.
