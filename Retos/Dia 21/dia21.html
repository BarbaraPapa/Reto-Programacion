Dia 21

Crea una lista desplegable donde se pueda seleccionar entre 3 grupos de productos ficticios.

Utilizamos la etiqueta < select > para crear la lista desplegable. Cada grupo está dentro de la etiqueta < optgroup > con un label. Cada opción dentro de la lista desplegable se define con la etiqueta < option >, donde especificamos el value.

Nota importante: el < select > generalmente se usa dentro de la etiqueta < form > cuando necesitamos recopilar los datos seleccionados por el usuario y enviarlos al servidor.
