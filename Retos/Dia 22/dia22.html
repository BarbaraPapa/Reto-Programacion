En la lista anterior (Dia 21), permite al usuario introducir opciones personalizadas (no sólo las de la lista) y filtrar las opciones existentes.

⬅ Volver

Es posible crear este tipo de control mediante la etiqueta < datalist >, utilicemos la etiqueta < input type="text" > para permitir al usuario introducir texto.

Nota importante: el < datalist > y el < input > generalmente se usan dentro de la etiqueta < form > cuando necesitamos recopilar los datos seleccionados por el usuario y enviarlos al servidor.
