Dia 5

Ponle un título y una descripción al documento, ideal para SEO.

La meta descripción es un atributo HTML que se utiliza para describir el contenido de una página y se inserta dentro del tag head .

<meta name="description" content="Este contenido fue creado para aprender HTML, en particular en esta página aprendemos cómo agregar una meta descripción ideal para SEO."/>

Puede aparecer en los resultados de búsqueda como una vista previa debajo del título para proporcionar información adicional.

Las metadescripciones no están hechas para contar historias. Cada palabra tiene su peso porque los usuarios dedican muy poco tiempo a elegir si hacer clic o no desplazándose rápidamente por la lista de resultados.
