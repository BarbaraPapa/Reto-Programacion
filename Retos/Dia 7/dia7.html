Coloca una imagen en formato JPEG-XL. Si el navegador no la soporta, que use AVIF. Sino, que use JPG.

⬅ Volver

Para solucionar este problema podemos utilizar la etiqueta < picture >. Dentro debemos usar la etiqueta < source >, que le indicará al navegador que intente leer la imagen si es compatible. También usaremos la etiqueta < img > que será la última que se compruebe.

< picture>
< source srcset="imagen.jpeg-xl" type="image/jxl" />
< source srcset="imagen.avif" type="image/avif" />
< img src="imagen.jpg" alt="Descripción de la imagen"/>
